#  Iris Data Analysis Using NumPy

##  Project Overview
This project analyzes the Iris flower dataset using Python and NumPy. The analysis includes data loading, preprocessing, statistical analysis, feature extraction, Boolean indexing, sorting, random sampling, and species-wise comparison without using Pandas.

---

##  Dataset Information

- Total Samples: 150
- Total Features: 4
- Total Species: 3

### Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Species
- Iris-setosa
- Iris-versicolor
- Iris-virginica

3. Install NumPy:

```bash
pip install numpy
```

4. Run all notebook cells.
